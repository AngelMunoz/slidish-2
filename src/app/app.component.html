What is Angular?

Angular is a Single Page Application (SPA) framework.

  • Backed by Google™.
  • One of the three most popular frameworks in the industry.
  • Uses components as its core element.
  • Has Object Oriented Programming features.
 What is Angular great at? (and what is not that great at)

The goodies.

  • Convention based.
  • Components at its core.
  • Well defined solutions to common problems.
       (routing, guards, services, pipes, directives)
  • Ahead of Time Compilation.
  • Excelent to work with teams.
  • Uses RxJs
  • Uses Typescript

The baddies.

  • Angular is verbose.
  • Relies heavily on Dependency Injection.
  • It makes it easy to generate big bundles.
  • Slow starting times.
  • Uses RxJs
  • Uses Typescript
